Especificações
Método de impressão Impressão a jato de tinta diretamente no tecido
Tamanho da máquina 137 cm x 104,5 cm x 66cm
Peso da máquina 111kg
Área de impressão máxima Padrão 35,5 cm x 40,6 cm (prancha inclusa)
Máx. 40,6 cm x 45,7 cm (c/download de firmware)
Tipo de tinta Pigmentada à base de água
Cores de tinta CMYK (GT-341); CMYK + Branco (GT-361/381)
Tipo de cabeça de impressão Piezo com gotejamento sob demanda
Núm. de cabeças de impressão 4 (GT-341); 6 (GT-361); 8 (GT-381)
Resolução de impressão 600 x 600 dpi, 1200 x 1200 dpi
Modo de impressão Passagem simples ou dupla para CMYK
Base e realce com tinta branca (GT-361/381)
Interfaces USB 2.0 de alta velocidade; LAN/Ethernet*
Condições ambientais Condições de impressão: 18°C a 30°C
Umidade relativa: 45-85% RH(sem condensação)
Fonte de alimentação 100V-240V CA, 50/60Hz 1Hz
Softwares
Requisitos de Windows Windows 8 (32bits/64bits), Windows7 (32 bit/64 bits), Windows Vista (32 bits/64 bits), Windows XP mais recente (32 bits)
Requisitos de interface e sistem operacional
  • Iterface Windows® Print Driver
  • Interface fácil, sem necessidade de RIP
  • Controle de volume da tinta branca na base e no realce (saturação, contraste, volume de tinta)(aplicável somente para GT-361/GT-381)
Condições de oper. do driver
  • 256 MB RAM (é recomendado 512 MB)
  • 128 MB de espaço disponível no HD
  • CPU de 1 GHz(é recomendado 2 GHz)
Aplicativos recomendados Adobe® Photoshop®, Adobe® Photoshop® Elements (para edição de bitmap), Adobe® Illustrator®, CorelDraw® (para edição de dados vetoriais), Corel® PaintShop™ Pro
Outros
Cabo USB Cabo USB 2.0 (tipo AB)
Memória Flash USB** Menos de 32 GB, formatação FAT32
Substrato recomendado 100% algodão (resultado final pode variar dependendo da mistura do material)
